[开灯 - 洛谷](https://www.luogu.com.cn/problem/P1876)

```c
//解 after 题解

#include <stdio.h>

int main()
{
	long long n, i;
	scanf("%lld", &n);
	for (i = 1; i * i <= n; i++)
		printf("%lld ", i * i);

	return 0;
}
```

只需枚举≤n的完全平方数即可

对于一个数n，其因子都是成对存在，所以同一盏灯必定会经过开关开关...最后关闭的状态，除非：它是个完全平方数

因为完全平方数=i\*i，而i=i，故其因子个数为奇数，最终为开启状态
