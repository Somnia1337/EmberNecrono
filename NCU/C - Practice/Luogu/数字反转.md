[[NOIP2011 普及组] 数字反转 - 洛谷](https://www.luogu.com.cn/problem/P1307)

```c
//解 after 题解

#include <stdio.h>

int main()
{
    int n, o = 0;
    scanf("%d", &n);
    for (; n >= 1 || n <= -1;)
    {
        o = o * 10 + n % 10;
        n /= 10;
    }
    printf("%d", o);

    return 0;
}
```

运用“余10再除10”，拆分后乘10，不仅快速巧妙，还适用于负数
