[小鱼的游泳时间 - 洛谷](https://www.luogu.com.cn/problem/P1425)

```c
#include<stdio.h>

int main()
{
    int a, b, c, d, x, y, M, H, n;
    scanf("%d %d %d %d", &a, &b, &c, &d);
    x = 60 * a + b;
    y = 60 * c + d;
    M = y - x;
    H = M / 60;
    M = M - 60 * H;
    printf("%d %d", H, M);
    return 0;
}
```
