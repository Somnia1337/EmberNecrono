---

内存中的变量存在 3 个区域：初始化的全局变量，未初始化的全局变量，局部变量。

32 位 OS 的指针占用 4 Byte。

```cpp
char c = 'a';
char* p = &c;
*p = 'b'; // <=> c = 'b'
```
