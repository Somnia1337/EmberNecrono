#### 3.20 ~ 3.28 🚫

#### 3.29 ✅

A.

```c
long sum = 0;
long i = 0;
while (i < 10) {
	if (i & 1)
		continue; // 无限循环
	sum += i;
	i++;
}
```

如果只是简单翻译到 `while`，当 `i` 首次为奇数时，`continue` 子句将导致 `i++` 更新子句被跳过，从而陷入死循环。

B.

```c
long sum = 0;
long i = 0;
while (i < 10) {
	if (i & 1)
		goto update;
	sum += i;
update:
	i++;
}
```
