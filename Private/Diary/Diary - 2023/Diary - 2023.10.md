血与泪的教训……把二维数组展开成一维应该是：

```java
for (int i = 0; i < rows * cols; i++)
{
	flat[i] = grid[i / cols][i % cols];
}
```

而不是：

```java
for (int i = 0; i < rows * cols; i++)
{
	flat[i] = grid[i / rows][i % cols];
```
